The `.map()` Method: The Makeover

The `.map()` method is your go-to for a total makeover. It takes an array, tries on a new look for *every single item*, and then gives you a brand new array with the transformed items. The original array stays untouchedâ€”so fierce!


const outfits = ["jeans", "skirt", "dress"];

// Give every outfit a glam makeover!
const glamOutfits = outfits.map(item => `sparkly ${item}`);

console.log(glamOutfits);
// Outputs: ["sparkly jeans", "sparkly skirt", "sparkly dress"]
console.log(outfits);
// Outputs: ["jeans", "skirt", "dress"] (Still the same!)

The `.filter()` Method: The VIP List

Think of `.filter()` as your personal party planner. You give it a list and a rule (like "only designer labels"), and it gives you back a new, exclusive list with only the items that made the cut. It's perfect for creating that VIP-only vibe.


const accessories = [
  { name: "purse", brand: "designer" },
  { name: "earrings", brand: "basic" },
  { name: "sunglasses", brand: "designer" }
];

// Only let the designer items on the list!
const designerOnly = accessories.filter(item => item.brand === "designer");

console.log(designerOnly);
// Outputs: [{ name: "purse", brand: "designer" }, { name: "sunglasses", brand: "designer" }]

The `.reduce()` Method: The Final Score

Sometimes you just need one final result from a whole collection. The `.reduce()` method takes all the items in your array and boils them down to a single value, like adding up the total cost of your shopping spree. It's the ultimate final score.


const shoppingSpree = [50, 25, 100]; // Prices of items

// Let's get the grand total.
const totalCost = shoppingSpree.reduce((sum, currentPrice) => sum + currentPrice, 0);

console.log(totalCost);
// Outputs: 175
